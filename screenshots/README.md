# Screenshots

This directory contains screenshots of the AWS Health Events Dashboard application.

## dashboard-main.png

Main dashboard screenshot showing:

- Status cards with health metrics
- Interactive filter bar
- Events table with 5 sample health events
- Dark theme UI with purple branding
- Real-time timestamp (Last updated 18:49:29)

**How to capture screenshots:**

1. Run the application: `npm run dev`
2. Open http://localhost:5173 in your browser
3. Take screenshots of different views and interactions
4. Save them in this directory with descriptive names

**Recommended screenshots:**

- dashboard-main.png (main view)
- dashboard-filtered.png (with filters applied)
- dashboard-mobile.png (mobile responsive view)
- event-detail.png (expanded event details)
