# Infrastructure as Code Templates

This directory contains Infrastructure as Code templates for deploying the AWS resources.

## CloudFormation

- Complete AWS resource definitions
- Nested stacks for modular deployment
- Parameter files for different environments

## Terraform (Alternative)

- Modular Terraform configurations
- State management setup
- Multi-environment support

## SAM (Serverless Application Model)

- Simplified serverless resource definitions
- Local testing capabilities
- Easy deployment with SAM CLI

Choose one approach based on your team's preferences and existing infrastructure patterns.
